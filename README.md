# Task Management Dashboard

Aplicație web construită cu React și TypeScript pentru managementul task-urilor. Aplicația permite afișarea unei liste de task-uri dintr-un API public, căutare după titlu, filtrare, sortare, afișarea detaliilor pentru task-ul selectat și operații de bază precum creare, editare și ștergere.

## Live Demo

https://task-management-dashboard-maria.netlify.app/

## Rulare proiect

```bash
npm install
```

Pornește aplicația în modul development:

```bash
npm run dev
```

Construiește aplicația pentru producție:

```bash
npm run build
```

Previzualizează build-ul de producție:

```bash
npm run preview
```
